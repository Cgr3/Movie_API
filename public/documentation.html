Search for movie by name /movies/[name] GET Specified movie by name JSON styled format of certain movie
Return data about a genre by name /movies/genre/[genre] GET Genre list from name JSON styled format of genre list
Return data about director by name /movies/director/[director] GET Director information by name JSON object of director information

Allow users to un-register /users/[id] DELETE Remove information of user data Deletion of user information
Allow users to add movies to favorites /users_movies PUT Add movies to their favorites tab JSON styled format of users favorite movies
Allow users to un-favorite movies /users_movies/[name] DELETE Remove specified movie from favorites JSON styled format of new user favorited movies
